# IPL2024_sql
This project analyzes IPL 2024 match data using SQL to uncover insights related to team performance, match outcomes, and competitive matches.

## Dataset
File: IPL 2024 -sheet1(1).csv
Records: 70+ matches
Key columns:
team1, team2
winner
result_margin
venue
date

# Tools Used
PostgreSQL / MySQL
Google Sheets (for visualization)
GitHub

# Key Analysis
1. Total Matches
Count of all matches played in IPL 2024
2. Team Performance
Total wins by each team
Matches played by each team
3. Close Matches
Matches decided by small margins (≤10 runs)
4. Toss Impact
Analysis of how toss influences match results

# Insights
Teams with strong batting lineups dominate wins
Toss plays a significant role in match outcomes
Close matches highlight competitive teams
